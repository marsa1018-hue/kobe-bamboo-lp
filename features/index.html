すべて1つのシステムの中でシームレスにつながっています
PHASE 1

仕込む

01 / SEARCH

商品検索・画像検索

Taobao・1688・T-mallの商品を、キーワードまたは画像でシステム内から直接検索。月間販売数・リピート率・取引評価も一覧で確認できます。

New
提供中
キーワードまたは画像で検索
販売数が多い順に並べて在庫切れリスクを下げる、価格帯で絞って粗利を改善するなど、目的に合わせて活用できます。粗利管理機能と連動し、いくらまでの仕入れなら利益が残るかを基準に仕入先を探せます。
↓
02 / PRICING

値付けツール

「いくらで売ればいくら残るか」を仕入れる前に固めます。

注目機能
提供中
フルコスト逆算で推奨売価・利益をその場でシミュレーション
商品代・代行手数料・国内運賃・プラットフォーム手数料を含めたフルコストから、目標粗利率を逆算して推奨売価をその場で算出。重量帯ごとの粗利も同時に試算できます。
↓
03 / LISTING

BASEへ出品（仕入れ先と紐づけ）

出品と同時に仕入れ先が記録され、以降の発注が自動化されます。

提供中
値付けが決まったらそのままBASEへ出品
出品と同時に「この商品はこの仕入れ先から仕入れる」という紐づけが記録されます。次回以降、受注が入るたびに自動で発注依頼が飛びます。
↓
PHASE 2

売れる

04 / ORDER

売れたら自動で発注依頼

注文が増えても、発注の手間は比例して増えません。

提供中
BASEで注文が入ると、紐づいた仕入れ先へ自動で発注
手作業ゼロ。注文確認から発注まで、システムが自動で処理します。受注が増えても作業量は変わりません。
↓
05 / SHIPPING

発送されたらショップに自動で発送通知

顧客への連絡も、追跡番号の入力も不要です。

提供中
追跡番号がBASEへ自動反映、エンドユーザーへ発送通知
仕入れ先から発送されると、追跡番号がBASEへ自動で反映されます。顧客への発送通知も完了するので、手動で連絡する必要がありません。

発送通知のタイミングを選べます。国際発送された時 or 日本で集荷された時。
PHASE 3

育てる

06 / PROFIT

粗利管理で実際の粗利を確認

利益が見える状態で、値付けや仕入れの判断ができます。

提供中
売上・コスト・運賃・手数料・粗利率をオーダー別・月次で自動集計
実際にいくら残っているかをオーダー単位・月次で把握。目標粗利率を設定しておくと、未達のオーダーが一目で分かります。
↓
07 / IMPROVE

目標粗利率に未達 → 改善提案

「売価を上げる」か「仕入れ単価を下げる」かを自動でサジェストします。

近日公開予定
未達オーダーをピックアップ → 改善のヒントを提示
目標に届かないオーダーを自動で抽出し、具体的な改善提案を表示。商品検索・値付けツールと連動して、すぐに次のアクションに移れます。
↓
08 / SWITCH

商品検索 → 値付け → 仕入れ先変更

次回から新しい仕入れ先で自動発注。切り替えも画面内で完結します。

提供中
商品紐付けを変更すれば、次回から新しい仕入れ先で自動発注
改善提案を受けて新しい仕入れ先を探し、値付けツールで粗利を確認。商品の紐付けを変更するだけで、次回注文から新しい仕入れ先へ自動で切り替わります。
09 / ACCOUNTING

月次データDL または 会計ソフト連携

経理の入力作業をなくしていきます。

そのまま税理士に渡せる。freee・マネーフォワードとも連携予定
月ごとのデータをワンクリックで出力し、そのまま税理士へ。会計ソフトとの連携も順次公開予定。経理の手間をシステムで削減します。
